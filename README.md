# Scientific Revolution Discoverers — אתר דו־לשוני

אתר סטטי בעברית ובאנגלית על:
גליליאו גליליי, ניקולאוס קופרניקוס, יוהאנס קפלר, טיכו ברהה, פרנסיס בייקון, רנה דקארט, אנדראס וסאליוס.

## קבצים
- `index.html` — דף האתר
- `styles.css` — עיצוב
- `app.js` — אינטראקטיביות, החלפת שפה, חלון מודאלי
- `data.js` — כל התוכן בעברית ובאנגלית
- `assets/discoveries/` — איורי SVG מוכנים של הגילויים
- `assets/people/` — כאן יש להוסיף תמונות דיוקן
